简易团队主页系统
======================================

## 使用说明

* 初次使用时先导入数据到mysql数据库
* `cp config-simple.php config.php`
* 修改配置文件填写数据库信息
* 手动修改数据库中setting的网站URL信息，不修改将导致页面跳转
* 进入后台修改导航链接

## 访问说明

* apache下需要PATH_INFO支持
* 或者使用伪静态规则，nginx下需提供重定向支持，已提供默认配置文件
